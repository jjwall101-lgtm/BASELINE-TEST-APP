# Together Steps - local test app

This test build uses the uploaded child-app layout and feature set.

## App identity

- App name: Together Steps
- Android package name: com.childparent.app
- Storage: local device/browser storage only
- Reward system: coins

## Included features

- Child Mode and Parent Mode
- Parent PIN
- Coin reward system
- Feelings page
- Rewards shop and reward requests
- Parent notes
- Quick daily log
- History and weekly reports
- Calendar
  - Child Mode: next 7 days
  - Parent Mode: current full month
- Visual timer
- Family tree tab
- Notifications via service worker
- Theme picker: Plain, Retro, Space, Minecraft-style, and Bunny
- Profile photo upload stored locally

## Not included

- Firebase
- Cloud sync
- Parent account sign-in
- Firestore rules
- Cross-device syncing

Saved data stays on the device/browser used for testing. Clearing browser/app data can delete the saved test data.
