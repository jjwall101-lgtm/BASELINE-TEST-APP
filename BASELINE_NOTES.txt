Together Steps - plain local coin test baseline

Built from the latest uploaded v51 baseline with Timer and Family tabs.
This version removes cloud sync and uses local device/browser storage only.
The test reward system uses coins.
No personal child image, personal names, or creator branding are included.


Theme pack added: Plain, Retro / Mario-style, Space, Minecraft-style, and Bunny. Firebase is still removed; saving remains local-only.

## Familiar child-app layout update

This test build keeps the Together Steps branding, local-only saving, coin reward system, profile photo upload, timer, calendar, family tree, and theme pack, but switches the interface back closer to the earlier child-app layout: bigger rounded cards, stronger child dashboard, chunky navigation, and the familiar traffic-light home view.
